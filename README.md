## DTC Datalandschap

## Doel van het project
Het doel van dit project is het ontwikkelen van een data-architectuur die een basis biedt voor het verzamelen, opslaan, verwerken en ontsluiten van data binnen de organisatie.

## Scope
Binnen dit project wordt gewerkt aan:
- Het in kaart brengen van databronnen.
- Het ontwerpen van een schaalbare data-architectuur.
- Het definiëren van datastromen en integraties.
- Het vastleggen van richtlijnen voor databeheer en governance.
